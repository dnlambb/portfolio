## Tech Stack

- Astro
- React
- Tailwind

## Run Locally

Clone the project

```bash
  git clone https://github.com/dnlambb/portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
